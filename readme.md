
# Car router

# Step 1

Install all packages with npm or yarn:

```npm install```

OR 

```yarn install```

# Step 2

Start server with node OR nodemon:

```node index.js```

OR

```nodemon index.js```


# Step 3

Server start's at http://localhost:8000
